# Pointers / Ponteiros / Zeiger

## 🇺🇸 English

This folder contains practical exercises developed during my studies of pointers in the C programming language.

Pointers are one of the most important concepts in C, allowing direct memory access, variable manipulation through addresses, dynamic memory usage, and efficient data handling.

### Topics practiced:
- Pointer declaration
- Memory addresses
- Dereferencing (`*`)
- Address operator (`&`)
- Arrays and pointers
- Pointer arithmetic
- Variable modification using pointers
- Function parameters with pointers
